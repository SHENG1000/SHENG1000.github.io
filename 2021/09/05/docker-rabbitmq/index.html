消息中间件-RabbitMQ

2021-09-05
144- 1m
漫漫长路 > JavaEE-Java消息中间件微服务SpringCloud

学习SpringCloud时要使用RabbitMQ消息中间件，使用docker安装到树莓派上，舒服

Docker安装

–>RabbitMQ官网

只能说爽死好吧

3.9版本

# for RabbitMQ 3.9, the latest series
docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.9-management

3.8版本

# for RabbitMQ 3.8,
# 3.8.x support timeline: https://www.rabbitmq.com/versions.html
docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.8-management

Win安装(不推荐)

  1. 安装Erlang
  2. 安装RabbitMQ
  3. 在Rabbit安装目录\sbin下执行指令，激活RabbitMQ's Management Plugin视图插件
rabbitmq-plugins enable rabbitmq_management
